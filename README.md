# wastewater-treatment-performance-monitoring
Python-based tool to monitor domestic wastewater treatment plant performance by analyzing COD, BOD, TSS and NH4 values. The program compares measured data with regulatory limits and visualizes results with graphs.
This project is a Python-based program developed to analyze the performance of a domestic wastewater treatment plant.

The program compares wastewater parameters such as COD, BOD, TSS and NH4 with regulatory discharge limits and visualizes results using graphs.

## Features
- Manual, automatic and CSV-based data input
- Performance efficiency calculation
- Regulatory limit control
- Graph visualization using Matplotlib

## Technologies
- Python
- Pandas
- Matplotlib

Run the program using:
